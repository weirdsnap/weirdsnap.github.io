# 常用容器（下）：字典、集合、元组、可选值、联合体

接上篇的队列、栈、堆，这篇我们覆盖 Python 里另一大类常用数据结构——键值对映射、集合运算、以及"有或者没有"的可选值。

---

## 哈希字典（最快键值对）

**抽象理解**：给每个键算个哈希值，直接定位到存储位置。插入、删除、查找平均 O(1)。键是无序的。

| 操作 | Python | C++ |
|------|--------|-----|
| 创建 | `d = {}` | `std::unordered_map<K, V> m;` |
| 设值 | `d[k] = v` | `m[k] = v` |
| 取值 | `d[k]` | `m[k]`（不存在会插入默认值） |
| 安全取值 | `d.get(k, default)` | `m.count(k)` 或 `m.find(k)` |
| 判断键存在 | `k in d` | `m.count(k) > 0` |
| 删除 | `del d[k]` | `m.erase(k)` |
| 大小 | `len(d)` | `m.size()` |
| 清空 | `d.clear()` | `m.clear()` |

```python
d = {}
d["a"] = 1
d["b"] = 2
print("a" in d)    # True
print(d.get("c", 0))  # 0
del d["a"]
```

{% raw %}
```cpp
#include <unordered_map>

std::unordered_map<std::string, int> m;
m["a"] = 1;
m["b"] = 2;
std::cout << m.count("a");       // 1（类似 in）
std::cout << m.count("c");       // 0

// 安全取值
auto it = m.find("c");
if (it != m.end()) {
    std::cout << it->second;
} else {
    std::cout << 0;
}

m.erase("a");
```
{% endraw %}

**常见操作对比**：

```python
# 遍历
for k, v in d.items():
    print(k, v)

# 只遍历键
for k in d:
    print(k)

# 只遍历值
for v in d.values():
    print(v)
```

```cpp
// 遍历键值对（C++17 结构化绑定）
for (const auto& [k, v] : m) {
    std::cout << k << " " << v << "\n";
}

// 只遍历键
for (const auto& [k, v] : m) {
    std::cout << k << "\n";
}

// 只遍历值
for (const auto& [k, v] : m) {
    std::cout << v << "\n";
}
```

**踩坑点**：

- C++ `m[k]` 如果键不存在，**会插入一个默认值**（int 是 0，string 是空串），然后返回它。这不是你想干的。安全写法是 `m.find(k)` 或 `m.count(k)`。
- `std::unordered_map` 遍历顺序无保证，取决于哈希函数和桶布局。不要依赖顺序。
- Python `dict` 3.7+ 保证插入顺序，C++ 没有直接对应物。如果需要插入顺序，用 `std::vector<std::pair<K, V>>` 手动维护。

## 有序字典（按键排序遍历）

**抽象理解**：Python 里 `sorted(d.items())` 得到的就是按键排序。C++ `std::map` 直接帮你排好了——它底层是红黑树，插入、删除、查找都是 O(log n)。

| 操作 | Python | C++ |
|------|--------|-----|
| 创建（插入顺序） | `d = {}` | 无直接对应 |
| 创建（键排序） | `sorted(d.items())` | `std::map<K, V> m;` |
| 设值 | `d[k] = v` | `m[k] = v` |
| 遍历顺序 | 插入顺序 | 键升序 |

```cpp
#include <map>

std::map<std::string, int> m;
m["c"] = 3;
m["a"] = 1;
m["b"] = 2;

for (const auto& [k, v] : m) {
    std::cout << k << " " << v << "\n";
}
// 输出：a 1, b 2, c 3（按键排序）
```

**什么时候用什么**：

| 需求 | Python | C++ |
|------|--------|-----|
| 最快增删查，不管顺序 | `dict` | `std::unordered_map` |
| 要按键顺序遍历 | `dict` + `sorted()` | `std::map`（自带排序） |
| 要插入顺序遍历 | `dict`（3.7+） | `std::vector<std::pair<K,V>>` |

## 插入顺序字典

**抽象理解**：记录每个键第一次插入的顺序，遍历时按那个顺序输出。

Python 3.7+ 的 `dict` 保证插入顺序。C++ 标准库没有直接等价物，手动实现有两种常见方式：

```cpp
// 方式一：vector + 辅助 set 去重
std::vector<std::pair<std::string, int>> ordered;
std::unordered_set<std::string> seen;

void set(std::string k, int v) {
    if (!seen.count(k)) {
        seen.insert(k);
        ordered.push_back({k, v});
    }
    // 注意：这里不处理更新已有键的顺序
}

// 方式二：boost::multi_index（第三方库）
```

## 哈希集合

**抽象理解**：一堆元素的集合，没有重复，也没有顺序。快速判断某个元素在不在里面。

| 操作 | Python | C++ |
|------|--------|-----|
| 创建 | `s = set()` | `std::unordered_set<T> s;` |
| 添加 | `s.add(x)` | `s.insert(x)` |
| 判断存在 | `x in s` | `s.count(x) > 0` |
| 删除 | `s.remove(x)` | `s.erase(x)` |
| 大小 | `len(s)` | `s.size()` |

```python
s = {3, 1, 2, 3}    # 重复的 3 被去重，s = {1, 2, 3}
s.add(4)
print(1 in s)        # True
print(len(s))        # 4
```

```cpp
#include <unordered_set>

std::unordered_set<int> s = {3, 1, 2, 3};  // 去重后 {3, 1, 2}
s.insert(4);
std::cout << s.count(1);   // 1（相当于 True）
std::cout << s.size();     // 4
```

**踩坑点**：

- Python `set` 自动去重，C++ `unordered_set` 也一样，重复插入不报错、不生效。
- 遍历顺序无保证，和 Python 的 `set` 一样。

## 有序集合

**抽象理解**：集合里的元素自动按大小排序。适合范围查询（"找大于 x 的所有元素"）。

```cpp
#include <set>

std::set<int> s = {3, 1, 2, 4};
for (int x : s) {
    std::cout << x << " ";   // 1 2 3 4
}

// 找大于 2 的第一个元素
auto it = s.upper_bound(2);
if (it != s.end()) {
    std::cout << *it;        // 3
}
```

| 对比 | Python | `std::unordered_set` | `std::set` |
|------|--------|----------------------|------------|
| 顺序 | 无保证 | 无保证 | 升序 |
| 查找 | O(1) 平均 | O(1) 平均 | O(log n) |
| 范围查询 | 不支持 | 不支持 | ✅ `lower_bound`/`upper_bound` |

## 集合运算

Python 的 `&`、`|`、`-` 直接做集合运算。C++ 需要手动循环或用 `<algorithm>`。

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a & b)   # {3}                  交集
print(a | b)   # {1, 2, 3, 4, 5}      并集
print(a - b)   # {1, 2}               差集
print(a ^ b)   # {1, 2, 4, 5}         对称差
```

```cpp
std::unordered_set<int> a = {1, 2, 3};
std::unordered_set<int> b = {3, 4, 5};

// 交集
std::unordered_set<int> inter;
for (int x : a) {
    if (b.count(x)) inter.insert(x);
}

// 并集
std::unordered_set<int> uni = a;
uni.insert(b.begin(), b.end());

// 差集
std::unordered_set<int> diff;
for (int x : a) {
    if (!b.count(x)) diff.insert(x);
}
```

如果用 `std::set`（有序），还可以用 `<algorithm>` 里的集合算法：

```cpp
std::set<int> a = {1, 2, 3};
std::set<int> b = {3, 4, 5};

std::set<int> result;
std::set_intersection(a.begin(), a.end(),
                      b.begin(), b.end(),
                      std::inserter(result, result.end()));
```

## 元组（异构固定结构）

**抽象理解**：把几个不同类型的东西打包成一个整体，编译期就知道有几个、各是什么类型。

| 操作 | Python | C++ |
|------|--------|-----|
| 创建 | `t = (1, "hello")` | `std::tuple<int, std::string> t(1, "hello")` |
| 两元素 | `t = (1, "hello")` | `std::pair<int, std::string> p = {1, "hello"}` |
| 取值 | `t[0]` | `std::get<0>(t)` |
| 解包 | `x, y = t` | `auto [x, y] = t`（C++17） |

```python
t = (1, "hello", 3.14)
print(t[0])      # 1
x, y, z = t      # 解包
```

```cpp
#include <tuple>

std::tuple<int, std::string, double> t = {1, "hello", 3.14};
std::cout << std::get<0>(t);   // 1
auto [x, y, z] = t;            // C++17 结构化绑定
```

**踩坑点**：

- C++ tuple 的索引必须是编译期常量，不能是变量。`std::get<i>(t)` 的 `i` 必须是 `constexpr`。
- 对于两个元素，用 `std::pair` 更简洁，`p.first` / `p.second` 比 `std::get` 直观。

```cpp
std::pair<int, std::string> p = {1, "hello"};
std::cout << p.first;    // 1
std::cout << p.second;   // "hello"
```

## 可选值

**抽象理解**：一个变量可能有一个值，也可能是"空"的。比用 `-1` 或 `nullptr` 表示"没有"更安全。

| 操作 | Python | C++ |
|------|--------|-----|
| 定义 | `v: Optional[int] = None` | `std::optional<int> v = std::nullopt;` |
| 赋值 | `v = 42` | `v = 42` |
| 判断有值 | `v is not None` | `if (v)` 或 `v.has_value()` |
| 取值（安全） | `v`（空会抛错） | `v.value()`（空抛 `std::bad_optional_access`） |
| 取值（默认值） | `v if v is not None else 0` | `v.value_or(0)` |

```python
def find(key):
    d = {"a": 1}
    return d.get(key)  # 不存在返回 None

v = find("b")
if v is not None:
    print(v)
```

```cpp
#include <optional>

std::optional<int> find(const std::string& key) {
    std::unordered_map<std::string, int> m = {{"a", 1}};
    auto it = m.find(key);
    if (it != m.end()) return it->second;
    return std::nullopt;
}

int main() {
    auto v = find("b");
    if (v) {
        std::cout << *v;
    } else {
        std::cout << 0;
    }
}
```

**踩坑点**：

- 不要用 `-1` 或 `nullptr` 表示"空"——它们可能是合法的值。`std::optional` 明确表示"可能没有值"。
- `v.value()` 在空时抛异常；`*v` 解引用在空时是未定义行为！

## 类型联合

**抽象理解**：一个变量在运行时可能是类型 A 或类型 B，但 C++ 要求你编译期就枚举所有可能性。

| 操作 | Python | C++ |
|------|--------|-----|
| 定义 | `v: Union[int, str] = 42` | `std::variant<int, std::string> v = 42;` |
| 切换 | `v = "hello"` | `v = "hello"` |
| 检查类型 | `isinstance(v, int)` | `std::holds_alternative<int>(v)` |
| 按类型取值 | 直接访问 | `std::get<int>(v)`（类型不对抛异常） |
| 统一访问 | 动态分发 | `std::visit([](auto&& val) { ... }, v)` |

```python
from typing import Union

v: Union[int, str] = 42
v = "hello"
if isinstance(v, str):
    print(v.upper())
```

```cpp
#include <variant>

std::variant<int, std::string> v = 42;
v = "hello";

// 方式一：按类型检查
if (std::holds_alternative<std::string>(v)) {
    std::cout << std::get<std::string>(v);
}

// 方式二：统一访问
std::visit([](auto&& val) {
    std::cout << val;
}, v);
```

**踩坑点**：

- C++ `std::variant` 的备选类型在编译期固定，不能在运行时动态新增类型。Python 的 `Union` 没有这个限制。
- `std::get<int>(v)` 如果 `v` 当前不是 int，抛 `std::bad_variant_access`。
- 空 `variant` 不存在——`std::variant` 默认用第一个类型的默认值构造。

## 快速对照

| 你需要 | Python | C++（头文件） |
|--------|--------|-------------|
| 哈希键值对 | `dict` | `std::unordered_map` (`<unordered_map>`) |
| 有序键值对 | `dict` + `sorted()` | `std::map` (`<map>`) |
| 插入顺序键值对 | `dict`（3.7+） | 手动 `vector<pair>` |
| 哈希集合 | `set` | `std::unordered_set` (`<unordered_set>`) |
| 有序集合 | `sorted(set)` | `std::set` (`<set>`) |
| 元组 | `tuple` | `std::tuple` / `std::pair` (`<tuple>`) |
| 可选值 | `Optional[T]` / `None` | `std::optional<T>` (`<optional>`) |
| 类型联合 | `Union[T, U]` | `std::variant<T, U>` (`<variant>`) |
