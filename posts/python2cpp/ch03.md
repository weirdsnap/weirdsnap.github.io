# 控制流：`if`、`while`、`for`、`switch`

控制流决定程序怎么跑。Python 和 C++ 在这部分长得很像，但几个关键点不一样，尤其是循环语法和条件判断的真假。

---

## `if` / `else`

### 基本写法

```python
score = 85
if score >= 90:
    print("A")
elif score >= 60:
    print("B")
else:
    print("C")
```

```cpp
int score = 85;
if (score >= 90) {
    std::cout << "A\n";
} else if (score >= 60) {
    std::cout << "B\n";
} else {
    std::cout << "C\n";
}
```

区别：

- Python 用 `elif`，C++ 用 `else if`
- C++ 的条件要放在括号里，语句块要放在大括号里
- C++ 的 `if (a = b)` 这种写法会悄悄赋值，是经典 bug

### 真假判断

Python 里很多东西都能当布尔值用：

```python
if "":       # False
if []:       # False
if 0:        # False
if None:     # False
if "hello":  # True
if [1, 2]:   # True
```

C++ 没这么宽松：

```cpp
if ("")      // 编译报错，const char* 不能隐式转 bool
if (0)       // false
if (nullptr) // false
if (s.empty()) // 要显式判断
```

C++ 里只有整数 0、浮点 0.0、`false`、`nullptr` 和能转这些的东西才会被认为是假。字符串、容器空不空，必须自己调 `.empty()` 或 `.size()` 判断。

## `while`

```python
n = 5
while n > 0:
    print(n)
    n -= 1
```

```cpp
int n = 5;
while (n > 0) {
    std::cout << n << "\n";
    --n;
}
```

基本一样，就是括号和大括号的区别。

Python 有 `while ... else`，C++ 没有：

```python
while n > 0:
    n -= 1
else:
    print("循环正常结束，没有被 break")
```

C++ 要实现类似效果，得用标志位：

```cpp
bool broken = false;
while (n > 0) {
    if (...) { broken = true; break; }
    --n;
}
if (!broken) std::cout << "循环正常结束\n";
```

## `for` 循环

这是两者差异最大的地方之一。

### Python：遍历可迭代对象

```python
for i in range(5):
    print(i)

for c in "hello":
    print(c)

for x in lst:
    print(x)
```

### C++：传统 `for`

```cpp
for (int i = 0; i < 5; ++i) {
    std::cout << i << "\n";
}
```

三个部分：`初始化; 条件; 步进`。都可以省略，但分号要保留。

### C++：范围 for

C++11 开始支持类似 Python 的写法：

```cpp
std::string s = "hello";
for (char c : s) {
    std::cout << c << "\n";
}

std::vector<int> v = {1, 2, 3};
for (int x : v) {
    std::cout << x << "\n";
}
```

如果要修改元素，加引用：

```cpp
for (int& x : v) {
    x *= 2;
}
```

### 带索引遍历

Python 常用 `enumerate`：

```python
for i, x in enumerate(lst):
    print(i, x)
```

C++ 范围 for 不直接给索引，需要手动计数：

```cpp
int i = 0;
for (const auto& x : v) {
    std::cout << i << " " << x << "\n";
    ++i;
}
```

或者回退到传统 `for`：

```cpp
for (size_t i = 0; i < v.size(); ++i) {
    std::cout << i << " " << v[i] << "\n";
}
```

## `switch` / `match-case`

Python 3.10+ 有 `match-case`：

```python
cmd = "start"
match cmd:
    case "start":
        print("starting")
    case "stop":
        print("stopping")
    case _:
        print("unknown")
```

C++ 用 `switch`：

```cpp
std::string cmd = "start";
// 注意：C++ switch 不能用 string，只能用整型、枚举、char
char c = 's';
switch (c) {
    case 's':
        std::cout << "starting\n";
        break;
    case 't':
        std::cout << "stopping\n";
        break;
    default:
        std::cout << "unknown\n";
}
```

C++ `switch` 有两个常见坑：

1. **只能用于整数类型**，不能直接用 `std::string`
2. **忘记 `break` 会贯穿到下一个 case**：

```cpp
switch (n) {
    case 1:
        std::cout << "one\n";
        // 忘写 break
    case 2:
        std::cout << "two\n";
        break;
}
// n=1 时会输出 one 和 two
```

C++17 有 `[[fallthrough]]` 属性可以显式说明你是故意贯穿的。

## 三元运算符

Python：

```python
result = "yes" if x > 0 else "no"
```

C++：

```cpp
std::string result = (x > 0) ? "yes" : "no";
```

C++ 三元运算符是 `? :`，注意优先级：

```cpp
// 这样写容易出问题
std::string s = x > 0 ? "yes" : "no";   // 可以
int a = 1 + x > 0 ? 10 : 20;            // 优先级陷阱，最好加括号
```

## `break`、`continue`、`pass`

| 作用 | Python | C++ |
|------|--------|-----|
| 跳出循环 | `break` | `break` |
| 跳过当前迭代 | `continue` | `continue` |
| 空语句占位 | `pass` | `;` 或空大括号 `{}` |

```python
if x > 0:
    pass   # 暂时没想好写什么
```

```cpp
if (x > 0) {
    ;      // 空语句
}
```

---

## 几个容易踩的坑

1. **C++ 的 `=` 和 `==` 不一样**

```cpp
if (x = 5) { ... }   // 把 5 赋值给 x，然后判断 x 为真，基本不是你想写的
if (x == 5) { ... }  // 判断 x 是否等于 5
```

2. **C++ `for` 里的变量只在循环内有效**

```cpp
for (int i = 0; i < 10; ++i) { ... }
// i 在这里不能用了
```

3. **Python 的 `for` 和 `else` 组合，C++ 没有**

4. **C++ `switch` 不能处理字符串**，字符串判断通常用 `if-else` 链或 map

---

## 快速对照

| 目标 | Python | C++ |
|------|--------|-----|
| 条件判断 | `if / elif / else` | `if / else if / else` |
| 循环条件 | `while cond:` | `while (cond) { ... }` |
| 遍历范围 | `for i in range(n)` | `for (int i = 0; i < n; ++i)` |
| 遍历容器 | `for x in lst` | `for (auto x : lst)` / `for (auto& x : lst)` |
| 多分支 | `match-case` | `switch-case`（限整数） |
| 三元运算 | `a if cond else b` | `cond ? a : b` |
| 跳出循环 | `break` | `break` |
| 跳过本次 | `continue` | `continue` |
| 空语句 | `pass` | `;` |
