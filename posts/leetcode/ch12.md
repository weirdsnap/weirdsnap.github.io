# 网格安全路径（2812. Find the Safest Path in a Grid）

## 题意

给定一个 `N × M` 的 0-1 网格 `grid`，其中 `1` 表示格子中有**小偷**，`0` 是空位。定义每个格子的**安全因子**为它到最近小偷的曼哈顿距离。路径的安全因子是路径上所有格子的安全因子的**最小值**。求从左上角 `(0,0)` 到右下角 `(N-1, M-1)` 的最大安全因子路径。

如果起点或终点本身就有小偷，安全因子为 0。

## 解法：多源 BFS + 最大堆 Dijkstra

### 思路

核心分两步：

1. **预处理**：先用多源 BFS 计算每个格子到最近的 `1` 的距离（安全因子）
2. **找最优路径**：用最大堆从起点走到终点，每次选择安全因子最大的方向扩展，路径的安全因子 = 路径上各格子安全因子的最小值

把最大安全因子转化为**路径瓶颈最大值**问题——类似"在图中找一条路径，使路径上最小边权最大"。

### 代码

```cpp
class Solution {
public:
    int maximumSafenessFactor(vector<vector<int>>& grid) {
        const int N = grid.size();
        const int M = grid[0].size();

        // 如果起点或终点有贼，安全因子为 0
        if (grid[0][0] == 1 || grid[N - 1][M - 1] == 1) {
            return 0;
        }

        // 第一步：多源 BFS，计算每个格子到最近 1 的距离
        vector<vector<int>> dist(N, vector<int>(M, -1));
        queue<pair<int, int>> q;
        for (int i = 0; i < N; i++) {
            for (int j = 0; j < M; j++) {
                if (grid[i][j] == 1) {
                    q.push({i, j});
                    dist[i][j] = 0;
                }
            }
        }

        while (!q.empty()) {
            auto [x, y] = q.front(); q.pop();
            int cur = dist[x][y];
            if (x > 0 && dist[x - 1][y] == -1) {
                q.push({x - 1, y});
                dist[x - 1][y] = cur + 1;
            }
            if (x < N - 1 && dist[x + 1][y] == -1) {
                q.push({x + 1, y});
                dist[x + 1][y] = cur + 1;
            }
            if (y > 0 && dist[x][y - 1] == -1) {
                q.push({x, y - 1});
                dist[x][y - 1] = cur + 1;
            }
            if (y < M - 1 && dist[x][y + 1] == -1) {
                q.push({x, y + 1});
                dist[x][y + 1] = cur + 1;
            }
        }

        // 第二步：最大堆找最优路径
        // 堆存 (安全因子, x, y)，安全因子大的优先
        priority_queue<tuple<int, int, int>> pq;
        vector<vector<bool>> visited(N, vector<bool>(M, false));
        pq.push({dist[0][0], 0, 0});

        while (!pq.empty()) {
            auto [safe, x, y] = pq.top(); pq.pop();

            if (x == N - 1 && y == M - 1) {
                return safe;
            }

            if (visited[x][y]) continue;
            visited[x][y] = true;

            // 四个方向扩展
            if (x > 0 && !visited[x - 1][y]) {
                pq.push({min(safe, dist[x - 1][y]), x - 1, y});
            }
            if (x < N - 1 && !visited[x + 1][y]) {
                pq.push({min(safe, dist[x + 1][y]), x + 1, y});
            }
            if (y > 0 && !visited[x][y - 1]) {
                pq.push({min(safe, dist[x][y - 1]), x, y - 1});
            }
            if (y < M - 1 && !visited[x][y + 1]) {
                pq.push({min(safe, dist[x][y + 1]), x, y + 1});
            }
        }

        return 0;
    }
};
```

### 算法步骤

**Step 1：多源 BFS**
- 从所有小偷（`grid[i][j] == 1`）同时出发，计算每个格子到最近小偷的曼哈顿距离
- 距离矩阵 `dist[i][j]` 就是该格子的安全因子
- 时间复杂度 O(N×M)

**Step 2：最大化最小路径值**
- 从 `(0,0)` 出发，优先队列按安全因子降序排列
- `safe` 记录当前路径上的最小安全因子，每走一步取 `min(safe, dist[新格子])`
- 第一次到达 `(N-1, M-1)` 时，`safe` 就是整条路径上安全因子的最大值

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(N×M log(N×M)) | BFS O(NM) + 堆扩展 O(NM log NM) |
| 空间 | O(N×M) | dist 矩阵 + visited + 堆 |

### 为什么是最大堆而不是最小堆？

这个问题的核心是**最大化路径上的最小值**。如果用 Dijkstra 的思路：

- 最小堆 Dijkstra 求最短路径（路径总和最小）
- **最大堆求路径瓶颈最大值**（路径上最小值最大）

每次从堆顶取出当前安全因子最大的节点，确保优先走"最安全"的格子——当第一次到达终点时，取到的就是所有路径中最小安全因子的最大值。

### 踩坑点 / 注意点

- ❌ 起点或终点有 1 要提前返回 0，否则 BFS 得到的 dist[0][0] = 0 也能走，但实际应该直接返回
- ❌ 多源 BFS 初始化时把所有 `1` 的位置入队，不是只从某个点出发
- ❌ 入队时就把 visited 标记为 true 防止重复入队（或者像上面那样在弹出时检查 `if (visited[x][y]) continue`）
- ❌ `priority_queue<tuple<int,int,int>>` 默认最大堆，tuple 按第一个元素降序排列，天然适合此题
