# 买卖股票的最佳时机 II（122. Best Time to Buy and Sell Stock II）

## 题意

给定一个数组 `prices`，`prices[i]` 是第 `i` 天的股票价格。你可以**多次买卖**股票，但同一时间只能持有一股。求能获得的最大利润。

```
输入: prices = [7,1,5,3,6,4]
输出: 7
解释: 
  第 2 天买入（1），第 3 天卖出（5），利润 4
  第 4 天买入（3），第 5 天卖出（6），利润 3
  总利润 4 + 3 = 7
```

---

## 解法：贪心，累加所有上涨

### 思路

因为可以无限次交易，所以只要今天比昨天价格高，就相当于可以在昨天买、今天卖，把这段利润吃掉。

换句话说：

```
总利润 = 所有相邻两天中，价格上涨的部分之和
```

比如 `[1, 5, 3, 6]`：
- `5 - 1 = 4` 涨，吃掉
- `3 - 5 = -2` 跌，不吃
- `6 - 3 = 3` 涨，吃掉
- 总利润 `4 + 3 = 7`

### 代码

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int last = prices[0];
        int result = 0;
        for (int i = 1; i < prices.size(); i++) {
            if (prices[i] > last) {
                result += prices[i] - last;
            }
            last = prices[i];
        }
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(N) | 只遍历一次数组 |
| 空间 | O(1) | 只用到两个变量 |

---

## 为什么贪心是对的

有人可能会想：是不是应该找"完整的上涨波段"，从最低点买到最高点卖，而不是每天小幅交易？

其实结果是一样的。

比如 `[1, 5, 3, 6]`：
- 波段操作：1 买 5 卖（赚 4），3 买 6 卖（赚 3），总利润 7
- 每天交易：1→5 赚 4，5→3 亏 2 不算，3→6 赚 3，总利润 7

波段操作的利润，等于把上涨波段拆成每一天上涨段的和。所以两者等价，而每天判断更简洁。

---

## 踩坑点

- ❌ 不要真的去模拟"买入-持有-卖出"的状态机，会写复杂
- ❌ 下跌时不要 subtract，只累加正利润
- ✅ 从第 1 天开始比较，第 0 天作为 `last` 初始值
- ✅ 如果 prices 为空或只有一天，利润为 0。题目保证至少一天
