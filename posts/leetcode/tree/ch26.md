# 二叉树的右视图（199. Binary Tree Right Side View）

## 题意

给定一棵二叉树的根节点 `root`，想象自己站在树的右侧，按照从顶部到底部的顺序，返回你能看到的节点值。

```
输入: [1,2,3,null,5,null,4]
输出: [1,3,4]

     1            <--- 看到 1
   /   \
  2     3         <--- 看到 3
   \     \
    5     4       <--- 看到 4
```

---

## 思路

### 关键观察

从右侧能看到的节点，恰好是**每一层最右边的节点**。问题转化为：层序遍历，每层取最后一个。

### 解法一：BFS 层序遍历

用队列实现 BFS，核心技巧是**按层处理**：

1. 遍历当前层之前，先记录队列大小 `size`
2. 只处理这 `size` 个节点（就是当前层的全部节点）
3. 处理过程中把下一层节点入队
4. 每层最后一个节点的值加入结果

### 解法二：DFS 先右后左

用栈模拟 DFS，**先压左子节点，再压右子节点**（这样右子节点先出栈）。记录当前访问到的最大深度，当遇到更深的节点时，就是该层第一个被访问到的节点（即最右节点）。

## 代码

### BFS

```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> result;
        queue<TreeNode*> q;
        if (root == nullptr) {
            return result;
        }
        q.push(root);
        
        while (!q.empty()) {
            // 当前层最右边的节点，就是队列尾部
            result.push_back(q.back()->val);
            
            // 按层处理：只处理当前层的 size 个节点
            int size = q.size();
            for (int i = 0; i < size; i++) {
                auto c = q.front();
                if (c->left != nullptr)
                    q.push(c->left);
                if (c->right != nullptr)
                    q.push(c->right);
                q.pop();
            }
        }
        return result;
    }
};
```

### DFS

```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> result;
        vector<pair<int, TreeNode*>> s;
        int cur_depth = 0;
        if (root == nullptr) {
            return result;
        }
        s.emplace_back(0, root);
        
        while (!s.empty()) {
            auto [depth, node] = s.back();
            s.pop_back();
            
            // 第一次到达这一深度，必然是最右节点
            if (depth >= cur_depth) {
                result.push_back(node->val);
                cur_depth++;
            }
            
            // 先压左，再压右，保证右先出栈
            if (node->left != nullptr) {
                s.emplace_back(depth + 1, node->left);
            }
            if (node->right != nullptr) {
                s.emplace_back(depth + 1, node->right);
            }
        }
        return result;
    }
};
```

---

## 关键细节

### BFS：取队尾 vs 取最后一个出队

两种写法都可以：

- **取队尾**（本代码）：`result.push_back(q.back()->val)`，在进入内层循环前取，此时队列里全是当前层节点，队尾就是最右。
- **取最后一个出队**：在内层循环中判断 `i == size - 1`，最后一个出队的节点就是最右。

取队尾更简洁，少一个条件判断。

### BFS：`size` 必须在循环前记录

```cpp
int size = q.size();  // 当前层节点数
for (int i = 0; i < size; i++)  // 只处理这么多
```

如果在 `for` 条件里写 `i < q.size()`，`q.size()` 会随着 `push` 子节点而变大，导致循环处理到下一层节点。

### DFS：为什么先压左再压右？

栈是 LIFO，**后压的先出**。先压左、再压右，出栈顺序就是**先右后左**。这样每一层第一个被访问到的节点就是最右边的节点。

### DFS：`depth >= cur_depth` 的含义

- `cur_depth` 表示结果数组 `result` 的当前长度（即已经收集了多少层）
- 当访问到一个深度 `depth` 大于等于 `cur_depth` 的节点时，说明这是第一次到达这一层
- 由于右子节点先被访问，这个节点必然是该层最右节点

---

## 复杂度

| 指标 | BFS | DFS | 说明 |
|------|-----|-----|------|
| 时间 | O(n) | O(n) | 每个节点访问一次 |
| 空间 | O(w) | O(h) | BFS 队列空间（w 为最大宽度），DFS 栈空间（h 为树高）|

---

## 踩坑点 / 注意点

- ❌ **空树处理**。`root == nullptr` 时直接返回空数组。
- ❌ **BFS：`q.back()` 的时机**。必须在 `for` 循环之前取，否则队列里会有下一层节点。
- ❌ **BFS：`q.size()` 在循环条件里动态变化**。必须先记录到局部变量 `size`。
- ❌ **DFS：压栈顺序**。必须先左后右，才能保证右子节点先出栈。
- ❌ **DFS：`cur_depth` 的更新**。只在 `depth >= cur_depth` 时更新，不要每次 `depth + 1`。
