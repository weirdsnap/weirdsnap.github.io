# 柱状图中最大的矩形（84. Largest Rectangle in Histogram）

## 题意

给定 `n` 个非负整数，表示柱状图中每个柱子的高度，每个柱子的宽度为 1。求在该柱状图中能够勾勒出来的矩形的最大面积。

## 解法：单调递增栈

### 思路

维护一个**单调递增栈**（栈底到栈顶高度递增）。当遇到比栈顶矮的柱子时，说明栈顶柱子的右边界确定了：

1. 弹出栈顶作为**矩形的高**
2. 新栈顶是**左边界**（第一个更矮的），当前柱子是**右边界**（第一个更矮的）
3. 矩形宽度 = `右下标 - 左下标 - 1`

**关键技巧**：遍历结束后，栈中剩余元素需要统一结算，右边界设为 `n`。

### 代码

```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        vector<int> inc; // 单调递增栈，存下标
        const int n = heights.size();
        int result = 0;

        for (int i = 0; i < n; i++) {
            int cur = heights[i];
            while (!inc.empty() && heights[inc.back()] >= cur) {
                int top = inc.back();
                inc.pop_back();
                int left = inc.empty() ? -1 : inc.back();
                result = max(result, (i - left - 1) * heights[top]);
            }
            inc.push_back(i);
        }

        // 结算栈中剩余元素，右边界为 n
        while (!inc.empty()) {
            int top = inc.back();
            inc.pop_back();
            int left = inc.empty() ? -1 : inc.back();
            result = max(result, (n - left - 1) * heights[top]);
        }
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素最多入栈出栈一次 |
| 空间 | O(n) | 栈空间 |

## 与接雨水（42）的对比

| | 42. 接雨水 | 84. 最大矩形 |
|--|-----------|-------------|
| **栈的性质** | 单调递减栈 | 单调递增栈 |
| **入栈条件** | 当前高度 < 栈顶高度 | 当前高度 > 栈顶高度 |
| **结算时机** | 当前高度 ≥ 栈顶高度 | 当前高度 ≤ 栈顶高度 |
| **弹出元素角色** | 凹槽底部（算积水面积） | 矩形高度（算矩形面积） |
| **宽度计算** | `i - left - 1` | `i - left - 1` |
| **边界处理** | 栈空即无左边界，无法积水 | 栈空即左边界为 -1，宽度为 `i` |
| **尾部处理** | 无需额外处理 | 需统一结算，右边界为 `n` |

## 踩坑点 / 注意点

- ❌ **栈里存的是下标，不是高度值**。计算宽度必须用下标差。
- ❌ **宽度不要忘记 `-1`**。两个下标 `i` 和 `j` 之间的宽度是 `j - i - 1`。
- ❌ **尾部结算不能漏**。遍历结束后栈中元素还没结算，右边界统一设为 `n`。
- ❌ **等号决定严格性**。`>=` 确保相邻等高柱子也能正确结算。
