> **标签**：单调栈
# 单调栈：接雨水 II & 柱状图中最大的矩形（42 / 84）

## 题意

### 42. 接雨水

给定 `n` 个非负整数表示每个柱子的宽度为 1 的高度，计算按此排列的柱子下雨之后能接多少雨水。

### 84. 柱状图中最大的矩形

给定 `n` 个非负整数，表示柱状图中每个柱子的高度，每个柱子的宽度为 1。求在该柱状图中能够勾勒出来的矩形的最大面积。

---

## 核心思想：单调栈

单调栈的本质是**维护一个具有单调性的序列**，利用栈的 LIFO 特性，在"破坏单调性"的时刻进行结算。

| 题型 | 栈的性质 | 结算时机 | 计算对象 |
|------|----------|----------|----------|
| 接雨水（42）| 单调递减栈 | 遇到更高的柱子 | 凹槽面积（积水）|
| 最大矩形（84）| 单调递增栈 | 遇到更矮的柱子 | 以弹出柱为高的矩形面积 |

两道题的核心差异：
- **接雨水**：找两边**更高**的柱子形成凹槽 → 递减栈，弹出矮的
- **最大矩形**：找两边**更矮**的柱子确定宽度 → 递增栈，弹出高的

---

## 42. 接雨水 — 单调递减栈

### 思路

维护一个**单调递减栈**（栈底到栈顶高度递减）。当遇到比栈顶高的柱子时，说明可以形成凹槽积水：

1. 弹出栈顶作为**凹槽底部**
2. 新栈顶是**左边界**，当前柱子是**右边界**
3. 积水高度 = `min(左, 右) - 底部`，宽度 = `右下标 - 左下标 - 1`

### 代码

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        const int length = height.size();
        vector<int> de_stack;
        int result = 0;
        int left, right, top, bottom;
        de_stack.push_back(0);  // index

        for (int i = 1; i < length; i++) {
            while (!de_stack.empty() && (height[i] >= height[de_stack.back()])) {
                bottom = height[de_stack.back()];
                de_stack.pop_back();
                if (de_stack.empty()) {
                    break;
                }
                left = height[de_stack.back()];
                right = height[i];
                top = std::min(left, right);
                result += (top - bottom) * (i - de_stack.back() - 1);
            }
            de_stack.push_back(i);
        }

        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素最多入栈出栈一次 |
| 空间 | O(n) | 栈空间 |

## 84. 柱状图中最大的矩形 — 单调递增栈

### 思路

维护一个**单调递增栈**（栈底到栈顶高度递增）。当遇到比栈顶矮的柱子时，说明栈顶柱子的右边界确定了：

1. 弹出栈顶作为**矩形的高**
2. 新栈顶是**左边界**（第一个更矮的），当前柱子是**右边界**（第一个更矮的）
3. 矩形宽度 = `右下标 - 左下标 - 1`

**关键技巧**：遍历结束后，栈中剩余元素需要统一结算，右边界设为 `n`。

### 代码

```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        vector<int> inc; // 单调递增栈，存下标
        const int n = heights.size();
        int result = 0;

        for (int i = 0; i < n; i++) {
            int cur = heights[i];
            while (!inc.empty() && heights[inc.back()] >= cur) {
                int top = inc.back();
                inc.pop_back();
                int left = inc.empty() ? -1 : inc.back();
                result = max(result, (i - left - 1) * heights[top]);
            }
            inc.push_back(i);
        }

        // 结算栈中剩余元素，右边界为 n
        while (!inc.empty()) {
            int top = inc.back();
            inc.pop_back();
            int left = inc.empty() ? -1 : inc.back();
            result = max(result, (n - left - 1) * heights[top]);
        }
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素最多入栈出栈一次 |
| 空间 | O(n) | 栈空间 |

## 两道题对比

| | 42. 接雨水 | 84. 最大矩形 |
|--|-----------|-------------|
| **栈的性质** | 单调递减栈 | 单调递增栈 |
| **入栈条件** | 当前高度 < 栈顶高度 | 当前高度 > 栈顶高度 |
| **结算时机** | 当前高度 ≥ 栈顶高度 | 当前高度 ≤ 栈顶高度 |
| **弹出元素角色** | 凹槽底部（算面积） | 矩形高度（算面积） |
| **宽度计算** | `i - left - 1` | `i - left - 1` |
| **边界处理** | 栈空即无左边界，无法积水 | 栈空即左边界为 -1，宽度为 `i` |
| **尾部处理** | 无需额外处理 | 需统一结算，右边界为 `n` |

---

## 踩坑点 / 注意点

- ❌ **栈里存的是下标，不是高度值**。计算宽度必须用下标差，不能直接用高度。
- ❌ **宽度不要忘记 `-1`**。两个下标 `i` 和 `j` 之间的宽度是 `j - i - 1`，不是 `j - i`。
- ❌ **84 的尾部结算不能漏**。遍历结束后栈中元素还没结算，右边界统一设为 `n`。
- ❌ **42 栈空时要 break**。没有左边界就无法形成凹槽，直接 break 继续。
- ❌ **等号的处理决定严格性**。`>=` 还是 `>` 影响相邻等高柱子的行为，两题都用 `>=` 确保正确结算。
