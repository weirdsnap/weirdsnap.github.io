# 接雨水（42. Trapping Rain Water）

## 题意

给定 `n` 个非负整数表示每个柱子的宽度为 1 的高度，计算按此排列的柱子下雨之后能接多少雨水。

## 解法一：动态规划（预处理左右最大值）

### 思路

对于每个位置 `i`，能接的雨水量取决于其左边最高柱子和右边最高柱子的较小值：

```
water[i] = min(left_max[i], right_max[i]) - height[i]
```

如果结果为正，说明该位置可以积水。

### 代码

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        const int length = height.size();
        // left max
        vector<int> left_max(length); 
        left_max[0] = height[0];
        for (int i = 1; i < length; i++) {
            left_max[i] = std::max(left_max[i-1], height[i]);
        }
        // right max
        vector<int> right_max(length);
        right_max[length - 1] = height[length - 1];
        for (int i = length - 2; i >= 0; i--) {
            right_max[i] = std::max(right_max[i+1], height[i]);
        }
        // result
        int result = 0;
        for (int i = 0; i < length; i++) {
            int temp = std::min(left_max[i], right_max[i]) - height[i];
            if (temp > 0) {
                result += temp;
            }
        }
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 三次线性扫描 |
| 空间 | O(n) | 两个辅助数组 |

## 解法二：双指针

### 思路

用两个指针从两端向中间移动，维护左右两侧遇到的最大高度。哪边小就先结算哪边，因为较小的一边决定了当前位置能接多少水。

### 代码

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        const int length = height.size();
        int left = 0;
        int right = length - 1;
        int leftmax = 0;
        int rightmax = 0;
        int result = 0;
        while (left < right) {
            // 左边小结算左边
            if (height[left] < height[right]) {
                if (height[left] >= leftmax) {
                    leftmax = height[left];
                } else {
                    result += leftmax - height[left];
                }
                left++;
            } else {
                if (height[right] >= rightmax) {
                    rightmax = height[right];
                } else {
                    result += rightmax - height[right];
                }
                right--;
            }
        }
        
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 单次遍历 |
| 空间 | O(1) | 只使用几个变量 |

## 解法三：单调栈

### 思路

维护一个单调递减栈。当遇到比栈顶高的柱子时，说明可以形成凹槽积水。弹出栈顶作为凹槽底部，计算新栈顶和新柱子之间的积水量。

### 代码

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        const int length = height.size();
        vector<int> de_stack;
        int result = 0;
        int left, right, top, bottom;
        de_stack.push_back(0);  // index

        for (int i = 1; i < length; i++) {
            while (!de_stack.empty() && (height[i] >= height[de_stack.back()])) {
                bottom = height[de_stack.back()];
                de_stack.pop_back();
                if (de_stack.empty()) {
                    break;
                }
                left = height[de_stack.back()];
                right = height[i];
                top = std::min(left, right);
                result += (top - bottom) * (i - de_stack.back() - 1);
            }
            de_stack.push_back(i);
        }

        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素最多入栈出栈一次 |
| 空间 | O(n) | 栈空间 |

## 解法对比

| 解法 | 时间 | 空间 | 特点 |
|------|------|------|------|
| 动态规划 | O(n) | O(n) | 思路直观，容易理解 |
| 双指针 | O(n) | O(1) | 空间最优，实现简洁 |
| 单调栈 | O(n) | O(n) | 通用框架，可解决接雨水 II |

## 踩坑点 / 注意点

- ❌ DP 解法里 `left_max` 和 `right_max` 的初始化容易写错。`left_max[i]` 应该包含 `height[i]` 本身，而不是 `i` 之前的最大值。
- ❌ 双指针中两边相等时结算右边，相等时两边都能结算，不影响结果。
- ❌ 单调栈里计算宽度时不要忘记 `-1`，积水宽度是 `i - de_stack.back() - 1`。
