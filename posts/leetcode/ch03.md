# 删除链表的中间节点（2095. Delete the Middle Node of a Linked List）

## 题意

给定一个链表的头节点 `head`，删除链表的中间节点，并返回修改后的链表的头节点。

- 如果节点总数为 `n`，中间节点是第 `⌊n / 2⌋` 个节点（从 0 开始计数）。
- 例如：`n = 5` 时删除第 2 个节点；`n = 4` 时删除第 2 个节点（0-based）。
- 当 `n = 1` 时，唯一节点即为中间节点，删除后链表为空，返回 `nullptr`。

## 解法一：快慢指针

### 思路

使用快慢指针：
- `slow` 每次走一步
- `fast` 每次走两步
- 当 `fast` 到达末尾时，`slow` 正好位于中间节点

为了删除中间节点，需要额外维护一个 `pre` 指针指向 `slow` 的前一个节点，然后执行 `pre->next = slow->next`。

### 代码

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* deleteMiddle(ListNode* head) {
        ListNode* pre = nullptr;
        ListNode* slow = head;
        ListNode* fast = head;
        while (fast->next != nullptr) {
            pre = slow;
            slow = slow->next;
            fast = fast->next;
            if (fast->next != nullptr) {
                fast = fast->next;
            } else {
                break;
            }
        }
        if (pre == nullptr)
            return nullptr;
        pre->next = slow->next;
        return head;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 快慢指针遍历约一半链表 |
| 空间 | O(1) | 只用了三个指针 |

## 解法二：快慢指针 + dummy 节点

### 思路

使用虚拟头节点 `dummy`，让 `slow` 和 `fast` 都从 `dummy` 出发。`fast` 先走两步，然后同步移动。这样循环结束时 `slow` 直接指向中间节点的前一个节点，不需要额外的 `pre` 指针。

### 代码

```cpp
class Solution {
public:
    ListNode* deleteMiddle(ListNode* head) {
        if (head == nullptr || head->next == nullptr) {
            return nullptr;
        }

        ListNode dummy(0, head);
        ListNode* slow = &dummy;
        ListNode* fast = &dummy;

        while (fast->next != nullptr && fast->next->next != nullptr) {
            slow = slow->next;
            fast = fast->next->next;
        }

        // slow 现在指向中间节点的前一个节点
        ListNode* mid = slow->next;
        slow->next = slow->next->next;
        delete mid;  // 可选，竞赛中通常不 delete

        return dummy.next;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 单次遍历 |
| 空间 | O(1) | 只使用指针 |

## 解法对比

| 解法 | 特点 |
|------|------|
| 快慢指针（原写法） | 从 `head` 出发，需要 `pre` 指针记录前一个节点 |
| 快慢指针 + dummy | 从虚拟头节点出发，`slow` 自然落在中间节点前一个位置，代码更简洁 |

## 踩坑点 / 注意点

- ❌ 忘记处理 `n = 1` 的情况 — 题目示例只给了 `n >= 2`，但提示中 `n` 的范围是 `[1, 10^5]`。当链表只有一个节点时，该节点本身就是中间节点，删除后链表为空，应返回 `nullptr`。很多同学第一次做这题时会在这里出错。
- ❌ `fast` 走两步时不判断 `fast->next` 是否为空 — 可能导致空指针解引用。

`n = 1` 时的输入输出：

```
输入：head = [1]
输出：[]
解释：n = 1，中间节点下标为 0，即头节点本身。删除后链表为空。
```
