# 最大子数组和（53. Maximum Subarray）

## 题意

给定一个整数数组 `nums`，找到一个**连续子数组**（至少包含一个元素），使其和最大，返回这个最大和。

```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6
```

---

## 思路

### 暴力想

枚举所有子数组，计算和，取最大。O(N³) 或 O(N²)，显然超时。

### 关键观察

假设我们已经知道以位置 `i-1` 结尾的最大子数组和是 `dp[i-1]`。那以 `i` 结尾的最大子数组和呢？

要么从 `i` 重新开始（前面的和是负的，不如不要），要么延续前面的子数组：

```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

这就是状态转移。`dp[i]` 表示**必须以 i 结尾**的连续子数组的最大和。

### 代码

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        // dp[i] 表示以 i 结尾的连续子数组最大和
        // 直接复用 nums 数组，省空间
        vector<int> dp(nums.begin(), nums.end());
        const int n = nums.size();
        
        // 同时维护全局最大值，避免第二次遍历
        int result = dp[0];
        
        for (int i = 1; i < n; i++) {
            // 状态转移：要么重新开始，要么延续前一个子数组
            dp[i] = max(dp[i], dp[i - 1] + nums[i]);
            // 实时更新全局最大值
            result = max(result, dp[i]);
        }
        
        return result;
    }
};
```

### 空间优化

`dp[i]` 只依赖 `dp[i-1]`，可以滚成 O(1)：

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int cur = nums[0];      // 以当前位置结尾的最大和
        int result = nums[0];   // 全局最大
        
        for (int i = 1; i < nums.size(); i++) {
            cur = max(nums[i], cur + nums[i]);
            result = max(result, cur);
        }
        
        return result;
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(N) | 单次遍历 |
| 空间 | O(1) | 只用一个变量 |

---

## 核心理解

`dp[i]` 的定义很关键：**必须以 i 结尾**。这保证了子数组的连续性——如果不要求连续，就变成"最大子序列和"，直接取所有正数相加就行。

`max(nums[i], dp[i-1] + nums[i])` 这个转移也很直观：前面的和如果是负的，`dp[i-1] + nums[i]` 比 `nums[i]` 还小，不如从 `i` 重新开始。

---

## 踩坑点

- ❌ `dp` 定义成"前 i 个元素的最大子数组和"，转移时会漏掉"必须连续"的约束
- ❌ 忘记处理全负数的情况，`result` 初始化为 `nums[0]` 而不是 0
- ❌ 空间优化时把 `cur` 和 `result` 初始化为 0，全负数数组会返回 0
- ✅ `dp[i]` 必须包含 `nums[i]`，这是连续性的保证
