# 正则表达式匹配（10. Regular Expression Matching）

> 本文先放一个原始版本，后续再优化思路和代码。

## 题意

给定一个字符串 `s` 和一个字符规律 `p`，实现一个支持 `'.'` 和 `'*'` 的正则表达式匹配。

- `'.'` 匹配任意单个字符
- `'*'` 匹配零个或多个前面的那一个元素

所谓匹配，是要涵盖整个字符串 `s` 的，而不是部分字符串。

## 原始解法

下面是第一版递归实现（JavaScript），后续会逐步优化：

```js
/**
 * @param {string} s
 * @param {string} p
 * @return {boolean}
 */
var isMatch = function (s, p) {
  var str = 0, ptr = 0;
  var slen = s.length, plen = p.length;
  function Match(str, ptr) {
    //console.log("s: "+str+" ; p: "+ptr);
    if (str > slen) return false;
    if (ptr > plen) return false;

    // 终止条件
    if (str == slen) {
      var gap = true;
      while (ptr != plen) {
        if (gap && p[ptr] != '*' || !gap && p[ptr] == '*') {
          ptr++;
        } else {
          break;
        }
        gap = !gap;
      }
      if (ptr == plen && gap) {
        return true;
      }
    } else {
      // 如果 p 的下一个值不是 '*'，直接比较当前值
      if (p[ptr + 1] != '*') {
        if (s[str] == p[ptr] || p[ptr] == '.') {
          return Match(str + 1, ptr + 1);
        } else {
          return false;
        }
      // 下一个是 '*'，匹配 0 到 n 个字符
      } else {
        var result = Match(str, ptr + 2);
        while (str < slen && p[ptr] == '.' || s[str] == p[ptr]) {
          result = result || Match(str + 1, ptr + 2);
          str++;
        }
        return result || Match(str, ptr + 2);
      }
    }
    return false;
  }
  var result = Match(str, ptr);
  return result;
};
```

## 待优化点

- 变量命名 `str` 外层和函数参数重名，容易混淆
- `s` 用完后的剩余模式判断可以更简单
- 递归有大量重复计算，可以加上记忆化
- 可以改写成自底向上的动态规划版本
