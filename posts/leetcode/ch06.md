# 寻找两个正序数组的中位数（4. Median of Two Sorted Arrays）

## 题意

给定两个已经排好序的数组 `nums1` 和 `nums2`，返回它们合并后的中位数。

- 总长度为奇数时，中位数是中间那个数
- 总长度为偶数时，中位数是中间两个数的平均值

要求时间复杂度为 `O(log(m+n))`，但双指针 `O(m+n)` 解法也能通过。

## 解法一：双指针遍历

### 思路

先分几种情况讨论：

1. **其中一个数组为空**：直接返回另一个数组的中位数。
2. **两个数组都不为空**：比较两个数组当前的元素，从相对较小的数组中拿取一个，重复进行。一共拿取 `(m + n) / 2 + 1` 个元素。若中途某个数组已经结束，则无需比较，直接从另一个数组拿取。
3. **最后按照 `m + n` 是否为奇数输出不同结果**：
   - 奇数时，最后一个取到的元素就是中位数
   - 偶数时，最后两个取到的元素的平均值就是中位数

### 代码

```cpp
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        // 处理其中一个数组为空的情况
        if (nums1.empty()) {
            int n = nums2.size();
            return n % 2 != 0
                ? nums2[n / 2]
                : (double)(nums2[n / 2 - 1] + nums2[n / 2]) / 2;
        }
        if (nums2.empty()) {
            int m = nums1.size();
            return m % 2 != 0
                ? nums1[m / 2]
                : (double)(nums1[m / 2 - 1] + nums1[m / 2]) / 2;
        }

        int len = nums1.size() + nums2.size();
        bool odd = len % 2 != 0;
        int target = 0;      // 当前取到的元素
        int lastone = 0;     // 上一次取到的元素
        int i = 0, j = 0;

        // 取前 len/2 + 1 个元素
        while (i + j < (double)len / 2 + 1) {
            lastone = target;
            if (i >= nums1.size()) {
                j++;
                target = nums2[j - 1];
            } else if (j >= nums2.size()) {
                i++;
                target = nums1[i - 1];
            } else if (nums1[i] < nums2[j]) {
                i++;
                target = nums1[i - 1];
            } else {
                j++;
                target = nums2[j - 1];
            }
        }

        if (odd) {
            return lastone;
        } else {
            return (double)(target + lastone) / 2;
        }
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m + n) | 最多遍历到中位数位置 |
| 空间 | O(1) | 只使用几个变量 |

## 解法二：二分查找（最优解）

### 思路

在两个数组中找到一个划分点，使得：
- 左半部分所有元素 <= 右半部分所有元素
- 左半部分和右半部分长度相等（偶数）或左边多一个（奇数）

在较短的数组上做二分，保证 `O(log(min(m, n)))`。

### 代码

```cpp
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        // 确保 nums1 是较短的数组
        if (nums1.size() > nums2.size()) {
            return findMedianSortedArrays(nums2, nums1);
        }
        
        int m = nums1.size();
        int n = nums2.size();
        int left = 0, right = m;
        
        while (left <= right) {
            int partition1 = (left + right) / 2;
            int partition2 = (m + n + 1) / 2 - partition1;
            
            int maxLeft1 = (partition1 == 0) ? INT_MIN : nums1[partition1 - 1];
            int minRight1 = (partition1 == m) ? INT_MAX : nums1[partition1];
            
            int maxLeft2 = (partition2 == 0) ? INT_MIN : nums2[partition2 - 1];
            int minRight2 = (partition2 == n) ? INT_MAX : nums2[partition2];
            
            if (maxLeft1 <= minRight2 && maxLeft2 <= minRight1) {
                if ((m + n) % 2 == 0) {
                    return (max(maxLeft1, maxLeft2) + min(minRight1, minRight2)) / 2.0;
                } else {
                    return max(maxLeft1, maxLeft2);
                }
            } else if (maxLeft1 > minRight2) {
                right = partition1 - 1;
            } else {
                left = partition1 + 1;
            }
        }
        
        return 0.0;  // 不会到达这里
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log(min(m, n))) | 在较短数组上二分 |
| 空间 | O(1) | 只使用几个变量 |

## 解法对比

| 解法 | 时间 | 空间 | 特点 |
|------|------|------|------|
| 双指针遍历 | O(m + n) | O(1) | 代码直观，容易理解，面试够用 |
| 二分查找 | O(log(min(m, n))) | O(1) | 时间最优，但边界处理复杂 |

## 踩坑点 / 注意点

- ❌ 数组为空时没有单独处理 — 虽然可以用统一逻辑处理，但单独处理空数组能让主循环更清晰。
- ❌ 循环次数算错 — 需要取 `len / 2 + 1` 个元素，才能保证奇偶两种情况都拿到所需的数。
- ❌ 奇偶返回逻辑反了 — 奇数时返回 `lastone`（中间那个），偶数时返回 `(target + lastone) / 2`。
- ❌ 二分划分时边界处理错误 — `partition1 == 0` 和 `partition1 == m` 时要用 `INT_MIN` / `INT_MAX` 作为哨兵。
