# 编辑距离（72. Edit Distance）

## 题意

给定两个字符串 `word1` 和 `word2`，计算将 `word1` 转换成 `word2` 所需的最少操作数。

可以对一个字符串执行以下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

## 解法一：递归（朴素 DFS，超时）

### 思路

定义 `helper(word1, word2, i, j)` 为将 `word1[0..i)` 转换为 `word2[0..j)` 的最小编辑距离。递归基和递推公式：

```
if i == 0: return j              // word1 已空，需要插入 j 个字符
if j == 0: return i              // word2 已空，需要删除 i 个字符

if word1[i-1] == word2[j-1]:     // 当前字符相等，无需操作
    return helper(i-1, j-1)

// 三种操作取最小值 + 1
del = helper(i-1, j)             // 删除 word1[i-1]
ins = helper(i, j-1)             // 插入 word2[j-1]
rep = helper(i-1, j-1)           // 替换 word1[i-1] 为 word2[j-1]
return 1 + min(del, ins, rep)
```

### 代码

```cpp
class Solution {
public:
    // 语义：计算 word1[0..i) 转换为 word2[0..j) 的最小编辑距离
    int helper(string &word1, string &word2, int i, int j) {
        if (i == 0) return j;  // word1 为空，需插入 j 个字符
        if (j == 0) return i;  // word2 为空，需删除 i 个字符

        if (word1[i-1] == word2[j-1]) {  // 当前字符相同
            return helper(word1, word2, i-1, j-1);
        }
        
        // 三种操作取最小值 + 1
        int del = helper(word1, word2, i-1, j);     // 删除 word1[i-1]
        int ins = helper(word1, word2, i, j-1);     // 插入 word2[j-1]
        int rep = helper(word1, word2, i-1, j-1);   // 替换 word1[i-1] 为 word2[j-1]
        return 1 + min({del, ins, rep});
    }
    
    int minDistance(string word1, string word2) {
        return helper(word1, word2, word1.size(), word2.size()); // 传入完整长度
    }
};
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(3^(m+n)) |
| 空间 | O(m+n) 栈深度 |

指数级复杂度，在 LeetCode 上会超时。但这是理解编辑距离问题的起点。

## 解法二：递归 + 记忆化搜索（自顶向下 DP）

### 思路

朴素递归中大量子问题被重复计算（例如不同的路径都可能到达同一对 `(i, j)` 状态）。用二维 `memo` 数组缓存结果，遇到已计算的直接返回。

### 代码

```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        vector<vector<int>> memo(m + 1, vector<int>(n + 1, -1));
        
        function<int(int, int)> dfs = [&](int i, int j) -> int {
            if (i == 0) {
                return j;
            }
            if (j == 0) {
                return i;
            }
            if (memo[i][j] != -1) {
                return memo[i][j];
            }
            
            if (word1[i-1] == word2[j-1]) {
                memo[i][j] = dfs(i-1, j-1);
                return memo[i][j];
            }
            
            int del = dfs(i-1, j);
            int ins = dfs(i, j-1);
            int rep = dfs(i-1, j-1);
            memo[i][j] = 1 + min({del, ins, rep});
            return memo[i][j];
        };
        
        return dfs(m, n);
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(mn) | 每个 `(i, j)` 状态只计算一次 |
| 空间 | O(mn) | memo 数组 + 递归栈 |

递归+记忆化本质上就是 DP，只是计算方向是自顶向下的。

## 解法三：二维动态规划（自底向上）

### 思路

定义 `dp[i][j]` 为 `word1[0..i)` 转换为 `word2[0..j)` 的最小编辑距离。

**状态转移**：

```
if word1[i-1] == word2[j-1]:
    dp[i][j] = dp[i-1][j-1]       // 字符相同，继承左上角结果
else:
    dp[i][j] = 1 + min(
        dp[i-1][j],               // 删除 word1[i-1]
        dp[i][j-1],               // 插入 word2[j-1]
        dp[i-1][j-1]              // 替换 word1[i-1] 为 word2[j-1]
    )
```

**边界条件**：
- `dp[0][j] = j`：`word1` 为空串，需要插入 j 个字符
- `dp[i][0] = i`：`word2` 为空串，需要删除 i 个字符

**三种操作的直观理解**：

| 操作 | dp 坐标变化 | 代价 |
|------|-------------|------|
| 删除 word1[i-1] | `dp[i-1][j]` → `dp[i][j]` | 1 |
| 插入 word2[j-1] | `dp[i][j-1]` → `dp[i][j]` | 1 |
| 替换 word1[i-1] → word2[j-1] | `dp[i-1][j-1]` → `dp[i][j]` | 1 |

**填表顺序**：从左到右、从上到下，保证当前格子依赖的左、上、左上三个方向都已计算。

### 代码

```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1));
        
        // 初始化边界
        for (int i = 0; i <= m; i++) dp[i][0] = i;  // 删除所有字符
        for (int j = 0; j <= n; j++) dp[0][j] = j;  // 插入所有字符
        
        // 填充 DP 表
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1[i-1] == word2[j-1]) {
                    dp[i][j] = dp[i-1][j-1];  // 无需操作
                } else {
                    dp[i][j] = 1 + min({
                        dp[i-1][j],    // 删除
                        dp[i][j-1],    // 插入
                        dp[i-1][j-1]   // 替换
                    });
                }
            }
        }
        return dp[m][n];
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(mn) | 两层循环填表 |
| 空间 | O(mn) | 二维 DP 表 |

## 解法四：一维 DP（滚动数组优化空间）

### 思路

观察二维 DP 的状态转移：计算 `dp[i][j]` 只依赖 `dp[i-1][j]`（上一行同列）、`dp[i][j-1]`（同行前列）、`dp[i-1][j-1]`（上一行前列）三个值。

因此只需要**一行 + 一个额外变量**：

- `dp[j]` 当前行第 j 列的值
- `prev` 存储 `dp[i-1][j-1]`（左上角）供下一轮使用

**核心技巧**：
- 遍历第 `i` 行前，`dp[0..n]` 中存储的是第 `i-1` 行的结果
- 从左向右遍历时，`dp[j-1]` 已经更新为第 `i` 行的值（相当于 `dp[i][j-1]`）
- `dp[j]` 更新前，先用 `temp = dp[j]` 保存旧值（即 `dp[i-1][j]`），之后这个 `temp` 会成为下一个 `j+1` 的 `prev`

**空间进一步优化技巧**：如果 `m < n`，交换两个字符串，确保 `dp` 的长度为 min(m, n)，降低空间开销。

### 代码

```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        if (m < n)
            return minDistance(word2, word1); // 确保 n <= m，空间 O(min(m,n))

        vector<int> dp(n + 1);
        iota(dp.begin(), dp.end(), 0); // 初始化 [0,1,2,...,n]

        for (int i = 1; i <= m; i++) {
            int prev = dp[0]; // 存储 dp[i-1][j-1]
            dp[0] = i;
            for (int j = 1; j <= n; j++) {
                int temp = dp[j]; // 保存当前 dp[j]（即将变为 dp[i-1][j]）
                if (word1[i - 1] == word2[j - 1]) {
                    dp[j] = prev;
                } else {
                    dp[j] = 1 + min(min(dp[j], dp[j - 1]), prev);
                }
                prev = temp; // 更新 prev 为下一个循环的 dp[i-1][j]
            }
        }
        return dp[n];
    }
};
```

### 复杂度

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(mn) | 两层循环，与二维 DP 相同 |
| 空间 | O(min(m, n)) | 仅一维数组 + 常数变量 |

## 解法对比

| 解法 | 时间 | 空间 | 特点 |
|------|------|------|------|
| 朴素递归 | O(3^(m+n)) | O(m+n) | 理解问题起点，超时 |
| 记忆化搜索 | O(mn) | O(mn) | 自顶向下，思考最自然 |
| 二维 DP | O(mn) | O(mn) | 自底向上，结构清晰 |
| 一维 DP | O(mn) | O(min(m,n)) | 空间最优，面试加分 |

## 踩坑点 / 注意点

- ❌ 递归写法传入 `(m, n)` 而非 `(m-1, n-1)` — 注意 `i` 和 `j` 是**长度**不是下标，比较时用 `word1[i-1] == word2[j-1]`
- ❌ `min({del, ins, rep})` 需要 C++11 的 `initializer_list` 重载，需要 `#include <algorithm>` 或直接 `<bits/stdc++.h>`，若用嵌套 `min(min(del, ins), rep)` 则无需头文件
- ❌ 一维 DP 中 `prev` 的更新顺序容易出错：必须先保存 `temp = dp[j]`，更新 `dp[j]`，再 `prev = temp`
- ❌ 忘记交换 m 和 n 来保证 `m >= n` — 虽然不影响正确结果，但空间不是最优
