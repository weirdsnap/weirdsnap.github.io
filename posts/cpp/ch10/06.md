# 常用位运算惯用法

位运算直接操作整数的二进制表示，熟练掌握几个惯用法可以写出紧凑、高效且无需分支的代码。

## 1. 判断 2 的幂

2 的幂二进制中只有一位是 1，因此 `x & (x - 1)` 会把最低位的 1 清零；若结果为 0，则原数是 2 的幂（要求 `x > 0`）。

```cpp
bool is_power_of_2(unsigned x) {
    return x && (x & (x - 1)) == 0;
}
```

## 2. 取最低位的 1 / 清零最低位的 1

- `x & -x` 只保留最低位的 1（其余位变 0）。
- `x & (x - 1)` 清零最低位的 1。

```cpp
unsigned lowbit = x & -x;          // 0001_0100 -> 0000_0100
unsigned cleared = x & (x - 1);    // 0001_0100 -> 0001_0000
```

## 3. 设置 / 清除 / 翻转某一位

```cpp
x |= (1u << n);     // 设置第 n 位
x &= ~(1u << n);    // 清除第 n 位
x ^= (1u << n);     // 翻转第 n 位
bool bit = (x >> n) & 1u;  // 读取第 n 位
```

## 4. 向上对齐到 2 的幂

把地址或大小 `sz` 向上取整到 `align` 的倍数，要求 `align` 为 2 的幂。

```cpp
size_t align_up(size_t sz, size_t align) {
    return (sz + align - 1) & ~(align - 1);
}
```

`~(align - 1)` 生成低位置 0 的掩码，配合前面的 `sz + align - 1` 实现向上取整。

## 5. 掩码取低 / 高字节

```cpp
uint8_t low  = x & 0xFFu;           // 低 8 位
uint8_t high = (x >> 24) & 0xFFu;   // 高 8 位（32 位整数）
```

## 6. 不用临时变量交换两数

```cpp
a ^= b;
b ^= a;
a ^= b;
```

实际工程中仍推荐 `std::swap`，但位运算交换展示了 `x ^ x = 0` 与 `x ^ 0 = x` 的性质。

## 7. 统计 1 的个数

C++20 起使用 `std::popcount`，编译器通常能映射到 CPU 的 `popcnt` 指令。

```cpp
#include <bit>
int ones = std::popcount(x);
```

GCC/Clang 也提供 `__builtin_popcount` 作为扩展。

## 常见误区

- 这些 trick 大多要求操作数是**无符号整数**；对有符号数进行移位和位运算可能触发实现定义行为或符号位问题。
- `align_up` 要求 `align` 是 2 的幂，否则结果错误。
- 不要为了“炫技”在所有地方使用位运算；可读性优先时，除法和条件判断往往更容易维护。
