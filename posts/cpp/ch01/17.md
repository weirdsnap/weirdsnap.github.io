# Lambda 按值捕获的 `const` 性

默认非 mutable Lambda 的 `operator()` 是 const 成员函数，按值捕获的变量表现为 const；`mutable` 解除限制。

**场景示例：**
```cpp
int x = 10;
auto f1 = [x]() { /* x 被视为 const int */ };
auto f2 = [x]() mutable { x = 20; /* 修改的是闭包内部副本 */ };
```

**原理解析 + 常见误区**

- ❌ 按值捕获的变量就是 const — 不精确。是闭包对象的成员变量在 const 成员函数中表现为 const。
- ❌ `mutable` 影响外部原变量 — 错误。`mutable` 只影响闭包内部副本。

**延伸点**：按值捕获指针时，指针本身是 const，但指向的内容不是——这与按值捕获对象有本质区别。
