# 获取类型的最大/最小值

C++ 使用 `<limits>` 头文件中的 `std::numeric_limits<T>` 获取各种算术类型的极限值，比 C 风格的宏（如 `INT_MAX`）更类型安全、更可扩展。

| 写法 | 含义 | 示例值 |
|------|------|--------|
| `std::numeric_limits<int>::max()` | `int` 能表示的最大值 | `2147483647` |
| `std::numeric_limits<int>::min()` | `int` 能表示的最小值 | `-2147483648` |
| `std::numeric_limits<unsigned>::max()` | 无符号类型最大值 | `4294967295` |
| `std::numeric_limits<double>::lowest()` | 最小的有限负值 | `-1.79769e+308` |
| `std::numeric_limits<double>::min()` | 最小的正规格化正值 | `2.22507e-308` |
| `std::numeric_limits<bool>::max()` | `true` | `1` |

### C 风格常量对照

| 类型 | C 风格常量（`<climits>` / `<cfloat>`） | `std::numeric_limits` 写法 |
|------|--------------------------------------|---------------------------|
| `int` | `INT_MAX` / `INT_MIN` | `numeric_limits<int>::max()` / `min()` |
| `long` | `LONG_MAX` / `LONG_MIN` | `numeric_limits<long>::max()` / `min()` |
| `long long` | `LLONG_MAX` / `LLONG_MIN` | `numeric_limits<long long>::max()` / `min()` |
| `unsigned int` | `UINT_MAX` | `numeric_limits<unsigned>::max()` |
| `double` | `DBL_MAX` / `DBL_MIN` | `numeric_limits<double>::max()` / `min()` |
| `float` | `FLT_MAX` / `FLT_MIN` | `numeric_limits<float>::max()` / `min()` |

```cpp
#include <iostream>
#include <limits>

int main() {
    std::cout << "int max: " << std::numeric_limits<int>::max() << "\n";
    std::cout << "int min: " << std::numeric_limits<int>::min() << "\n";
    std::cout << "size_t max: " << std::numeric_limits<size_t>::max() << "\n";
    std::cout << "double lowest: " << std::numeric_limits<double>::lowest() << "\n";
    std::cout << "double min: " << std::numeric_limits<double>::min() << "\n";
}
```

**原理解析 + 常见误区**

- ❌ `std::numeric_limits<T>::min()` 对所有类型都是最小值 — 错误。对浮点类型，`min()` 返回最小的**正**规格化值；要获取最小负值用 `lowest()`（C++11）。
- ❌ `INT_MAX` 比 `numeric_limits` 更好 — 错误。`numeric_limits` 与类型绑定，模板代码中更通用；且 `INT_MAX` 这类宏没有类型检查。
- ❌ 自定义类型不能用 `numeric_limits` — 错误。可以通过特化 `std::numeric_limits<MyType>` 为自定义类型提供极限值。

**延伸点**：`std::numeric_limits<T>::digits` 表示尾数/有效数字位数，`epsilon()` 表示 1 与下一个可表示值的差距，是浮点比较和数值算法中的常用工具。
