# 函数参数求值顺序未指定

C++ 不指定函数各参数的求值顺序，但保证同一参数的求值不会被其他参数打断（不交错）。

**场景示例：**
```cpp
int i = 0;
foo(++i, ++i);  // 未指定行为：可能是 foo(1, 2) 或 foo(2, 1)
```

**常见误区**

- ❌ 函数参数求值顺序是从左到右 — 错误。C++ 不保证顺序。
- ❌ 这是未定义行为（UB）— 错误。这是未指定行为（Unspecified Behavior），结果不确定但程序不会崩溃。

**延伸点**：C++17 起 `a.b`、`a->b`、`a(b)` 等少数运算符有顺序保证，但普通函数调用参数顺序仍未指定。
