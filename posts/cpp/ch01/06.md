# `auto` 的类型推导

`auto` 遵循模板按值推导规则：剥离顶层 `const`/`volatile` 和引用，数组与函数退化为指针。

| 表达式 | `auto` 推导结果 |
|--------|----------------|
| `const int` | `int`（顶层 const 剥离） |
| `int&` | `int`（引用剥离） |
| `int[5]` | `int*`（数组退化） |
| `{1, 2}` | `std::initializer_list<int>`（花括号特殊规则） |

**原理解析 + 常见误区**

- ❌ `auto` 保留引用 — 错误。`auto a = ref;` 中 `a` 是独立的 `int` 副本。
- ❌ `auto& b = x;` 能绑定 `const int` — 错误。`auto` 推导为 `int`，`b` 变成 `int&`，无法绑定 `const int`。

**延伸点**：`decltype(auto)`（C++14）使用 `decltype` 的推导规则替代 `auto` 的退化规则，可保留引用和 cv 限定符。
