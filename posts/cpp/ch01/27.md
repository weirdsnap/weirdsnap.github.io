# std::string 的三种操作风格

C++ 里处理字符串至少有三种写法：C 风格、成员函数/操作符重载、STL 算法风格。它们都能完成类似的事，但语法和适用场景不同。本章把同一类操作用三种方式各写一遍，方便你按需选择。

---

## 1. C 风格：`<cstring>` 函数族

C 风格直接操作 `char*` 或 `char[]`，需要手动管理 `\0` 结尾和缓冲区大小。

### 常见操作

```cpp
#include <cstring>
#include <iostream>

char s1[64] = "hello";
char s2[64] = "world";
char dest[128];

// 拷贝
std::strcpy(dest, s1);           // dest = "hello"
std::strncpy(dest, s1, 3);       // dest = "hel"（不保证补 \0）

// 拼接
std::strcat(dest, " ");
std::strcat(dest, s2);           // dest = "hello world"

// 长度
std::size_t len = std::strlen(dest);

// 比较
int cmp = std::strcmp(s1, s2);   // <0, 0, >0

// 查找
const char* p = std::strstr(dest, "world");  // 找子串
```

### 特点

| 优点 | 缺点 |
|------|------|
| 零开销、兼容 C 接口 | 手动管理内存和 `\0` |
| 在嵌入式或底层代码中常见 | 非常容易缓冲区溢出 |
| 适合和 C API 交互 | 不支持现代 C++ 的 RAII 和异常安全 |

> 现代 C++ 代码里应尽量避免直接用 C 风格字符串做业务逻辑，除非必须对接 C 接口。

---

## 2. 成员函数 + 操作符重载：`std::string`

`std::string` 把字符串封装成对象，提供类似 Python/Java 字符串的直观接口。

### 常见操作

```cpp
#include <string>
#include <iostream>

std::string s = "hello";

// 拼接
s = s + " world";          // 操作符重载
s += "!";                  // 追加
s.append("!!");            // 成员函数
s.push_back('?');          // 加一个字符

// 子串
std::string sub = s.substr(0, 5);   // 从 0 开始，取 5 个字符

// 查找
auto pos = s.find("world");         // 返回位置或 std::string::npos

// 插入、删除、替换
s.insert(5, " C++");       // 在位置 5 插入
s.erase(5, 4);             // 从位置 5 开始删 4 个字符
s.replace(0, 5, "Hi");     // 把 [0,5) 替换成 "Hi"

// 长度与访问
std::cout << s.size();     // 长度
std::cout << s[0];         // 访问字符（不检查边界）
std::cout << s.at(0);      // 访问字符（越界抛异常）
s[0] = 'h';                // 直接修改字符
```

### 特点

| 优点 | 缺点 |
|------|------|
| 语法直观，接近高级语言 | 每个操作是成员函数，得记 API 名字 |
| 自动管理内存，安全 | `find` 返回 `npos`，初学者容易忽略 |
| 支持修改单个字符 | 大量头尾操作时效率不如 `std::string_view` |

这是日常写 C++ 业务代码最常用的风格。

---

## 3. STL 风格：迭代器 + 算法

STL 风格把字符串当成一个元素序列，用迭代器和 `<algorithm>` 里的通用算法操作它。

### 常见操作

```cpp
#include <string>
#include <algorithm>
#include <iostream>

std::string s = "hello world";

// 查找
auto it = std::find(s.begin(), s.end(), 'w');

// 查找子串位置
auto it2 = std::search(s.begin(), s.end(), "world", "world" + 5);

// 反转
std::reverse(s.begin(), s.end());   // "dlrow olleh"

// 排序字符
std::sort(s.begin(), s.end());      // 按 ASCII 排序

// 拷贝
std::string dest;
dest.resize(s.size());
std::copy(s.begin(), s.end(), dest.begin());

// 统计
int cnt = std::count(s.begin(), s.end(), 'l');

// 替换所有某字符
std::replace(s.begin(), s.end(), 'l', 'L');

// 判断回文
bool is_palindrome = std::equal(s.begin(), s.begin() + s.size() / 2, s.rbegin());
```

### 特点

| 优点 | 缺点 |
|------|------|
| 通用，同一套算法可用于 vector/list 等 | 写起来比成员函数长 |
| 强调“对区间做操作”的思维方式 | 子串、插入、删除等操作不如 `std::string` 成员函数直观 |
| 配合 lambda 可以做复杂条件 | 调试时错误信息比较晦涩 |

---

## 三种风格对比：同一操作的三种写法

| 操作 | C 风格 | `std::string` 成员函数 | STL 算法 |
|------|--------|------------------------|----------|
| 拼接 | `strcat(dest, src)` | `s += src` / `s.append(src)` | `std::copy` + `back_inserter` |
| 取长度 | `strlen(s)` | `s.size()` / `s.length()` | `std::distance(s.begin(), s.end())` |
| 查找字符 | `strchr(s, 'a')` | `s.find('a')` | `std::find(s.begin(), s.end(), 'a')` |
| 查找子串 | `strstr(s, sub)` | `s.find(sub)` | `std::search` |
| 反转 | 手动循环 | 无直接成员函数 | `std::reverse` |
| 排序字符 | `qsort` | 无直接成员函数 | `std::sort` |
| 拷贝 | `strcpy` / `strncpy` | `s2 = s1` | `std::copy` |

---

## 该用哪种

| 场景 | 推荐风格 |
|------|----------|
| 和 C API 交互、嵌入式限制 | C 风格（`<cstring>`） |
| 日常业务代码、读写字符串 | `std::string` 成员函数 + 操作符 |
| 需要对字符序列做通用算法 | STL 算法 + 迭代器 |
| 只读访问大量字符串 | `std::string_view`（零拷贝） |

## 一个综合例子

把 `"  hello world  "` 去掉首尾空格，再转成大写：

```cpp
#include <string>
#include <algorithm>
#include <cctype>
#include <iostream>

std::string trim_and_upper(const std::string& s) {
    auto left = std::find_if(s.begin(), s.end(),
                             [](unsigned char c) { return !std::isspace(c); });
    auto right = std::find_if(s.rbegin(), s.rend(),
                              [](unsigned char c) { return !std::isspace(c); }).base();

    std::string result(left, right);
    std::transform(result.begin(), result.end(), result.begin(),
                   [](unsigned char c) { return std::toupper(c); });
    return result;
}

int main() {
    std::string s = "  hello world  ";
    std::cout << trim_and_upper(s);   // "HELLO WORLD"
}
```

这个例子同时用了 STL 算法和 `std::string` 构造，是实际代码里最常见的混合用法。
