# `decltype` 的推导规则

`decltype(e)`：标识符表达式推导为声明类型，其他表达式按值类别推导（左值→`T&`，将亡值→`T&&`，纯右值→`T`）。

**场景示例：**
```cpp
int i = 42;
decltype(i)   a = i;   // int
decltype((i)) b = i;   // int&（表达式语境，i 是左值）
```

**原理解析 + 常见误区**

- ❌ 表达式都推导为左值引用 — 错误。左值表达式推导为 `T&`，将亡值推导为 `T&&`，纯右值推导为 `T`。
- ❌ `decltype(auto)` 和 `auto` 结果相同 — 错误。`auto` 退化，`decltype(auto)` 保留引用和 cv。

**延伸点**：`decltype(auto) x = (local);` 推导为左值引用，若 `local` 是局部变量，返回该引用将导致悬垂。
