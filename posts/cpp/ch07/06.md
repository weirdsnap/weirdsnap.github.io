# std::format 类型安全格式化

C++20 的 `std::format` 提供编译期格式字符串检查，类型不匹配直接编译失败，语法接近 Python 的 f-string，是 `printf` 的现代替代。

| 特性 | `printf` | `std::format` | `cout <<` |
|------|----------|---------------|-----------|
| 类型安全 | ❌ | ✅ | ✅ |
| 格式控制 | 字符串内 | 字符串内 | 流操纵器 |
| 性能 | 快 | 快 | 慢（虚函数） |
| 编译期检查 | 部分警告 | ✅ | N/A |

```cpp
#include <format>
#include <iostream>

int main() {
    // 类型安全：编译期检查
    std::string s = std::format("Error {}: {}\n", 404, "Not Found");
    
    // 对齐、精度、进制
    std::cout << std::format("{:>10.2f}\n", 3.14159);   // "      3.14"
    std::cout << std::format("{:04x}\n", 255);          // "00ff"
    std::cout << std::format("{:*^10}\n", "hi");       // "***hi****"
    
    // 以下代码编译失败：
    // std::format("{:d}", 3.14);  // 格式说明符要求整数
}
```

**原理解析 + 常见误区**

- ❌ `std::format` 性能不如 `printf` — 错误。现代实现通常与 `printf` 相当甚至更快。
- ❌ `std::format` 不支持自定义类型 — 错误。可通过特化 `std::formatter<T>` 扩展任意类型。

**延伸点：** `std::format_to` 可将结果写入迭代器而非返回字符串，避免额外分配。自定义类型可通过特化 `std::formatter<T>` 支持格式化。
