# `std::cout` 默认输出类型

`std::cout` 通过 `operator<<` 重载支持内置算术类型、指针、C 字符串及部分标准库类型。

| 类型 | 默认输出形式 | 陷阱 |
|------|-------------|------|
| `bool` | `0` / `1` | 不是 `true`/`false`；需 `std::boolalpha` |
| `char` | **单个字符** | 不是 ASCII 数值；需显式转 `int` |
| `char*` | C 字符串 | 想输出地址必须显式转 `void*` |
| `std::nullptr_t` | 未定义/编译错误 | 应转 `const void*` |

**原理解析 + 常见误区**

- ❌ `char` 输出 ASCII 码 — 错误。`std::cout << 'a'` 输出字符 `a`。
- ❌ `std::cout << nullptr` 输出地址 — 错误。`std::nullptr_t` 没有标准 `operator<<` 重载。

**延伸点**：自定义 `operator<<` 的标准签名：`std::ostream& operator<<(std::ostream& os, const T& obj)`，返回 `ostream&` 支持链式调用。
