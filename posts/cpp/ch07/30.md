# std::sort：不只是快排

## 基本用法

```cpp
#include <algorithm>
#include <vector>

std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};
std::sort(v.begin(), v.end());           // 升序
std::sort(v.begin(), v.end(), std::greater<int>());  // 降序
```

## 自定义比较

```cpp
// 1. lambda
std::sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // 降序
});

// 2. 函数对象
struct Compare {
    bool operator()(int a, int b) const {
        return a < b;
    }
};
std::sort(v.begin(), v.end(), Compare());

// 3. greater<>（C++14 起支持模板参数推导）
std::sort(v.begin(), v.end(), std::greater<>());
```

## 底层实现：Introsort

`std::sort` 不是纯快排，而是 **Introsort**（Intropective Sort）：

1. **主体是快排** — 平均 O(N log N)
2. **递归深度超过阈值时切换到堆排** — 避免快排最坏 O(N²)
3. **小区间（通常 <= 16 个元素）用插入排序** — 缓存友好

| 算法 | 平均 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|
| 快排 | O(N log N) | O(N²) | O(log N) | ❌ |
| 堆排 | O(N log N) | O(N log N) | O(1) | ❌ |
| 插入排序 | O(N²) | O(N²) | O(1) | ✅ |
| **Introsort** | **O(N log N)** | **O(N log N)** | **O(log N)** | ❌ |

## 稳定性

`std::sort` **不稳定**。如果需要稳定排序，用 `std::stable_sort`：

```cpp
std::stable_sort(v.begin(), v.end());  // O(N log² N) 或 O(N log N)，稳定
```

## 部分排序

```cpp
// 只排序前 3 个元素
std::partial_sort(v.begin(), v.begin() + 3, v.end());

// 找第 k 小元素，不保证两边有序
std::nth_element(v.begin(), v.begin() + k, v.end());
```

## 踩坑点

- ❌ 比较函数不满足严格弱序（Strict Weak Ordering）— 会导致未定义行为
- ❌ 比较函数里做复杂计算或副作用 — 排序过程中会调用多次
- ❌ 对链表用 `std::sort` — 链表用 `list::sort`
- ✅ 自定义类型记得重载 `operator<` 或提供比较器
- ✅ C++20 起可以用 `std::ranges::sort`，支持管道语法

## 延伸

- `std::stable_sort` — 稳定排序
- `std::partial_sort` — 部分排序
- `std::nth_element` — 快速选择
- `std::binary_search` / `std::lower_bound` / `std::upper_bound` — 二分查找（需要先排序）
