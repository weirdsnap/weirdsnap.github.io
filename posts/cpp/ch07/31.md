# 二分查找：手动实现 vs STL

## 手动实现

最基础的二分查找，找目标值的位置：

```cpp
int binarySearch(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防溢出
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return -1;  // 没找到
}
```

### 边界陷阱

| 写法 | 循环条件 | 收缩方式 | 适用场景 |
|------|---------|---------|---------|
| `left <= right` | `left = mid + 1` / `right = mid - 1` | 查找具体值 | 最常用 |
| `left < right` | `left = mid + 1` / `right = mid` | 查找左边界 | 需处理 |
| `left + 1 < right` | `left = mid` / `right = mid` | 查找插入位置 | 开区间 |

最容易错的是 `mid` 的计算和边界收缩。`left + (right - left) / 2` 比 `(left + right) / 2` 安全，防溢出。

## STL 提供

C++ 标准库在 `<algorithm>` 里提供了四个二分查找工具：

| 函数 | 功能 | 返回值 | 前提条件 |
|------|------|--------|---------|
| `lower_bound` | 第一个 **≥** val 的位置 | 迭代器 | 范围已排序 |
| `upper_bound` | 第一个 **>** val 的位置 | 迭代器 | 范围已排序 |
| `equal_range` | 同时返回 lower 和 upper | pair<迭代器, 迭代器> | 范围已排序 |
| `binary_search` | 判断 val 是否存在 | bool | 范围已排序 |

### lower_bound

```cpp
#include <algorithm>
#include <vector>

vector<int> v = {1, 2, 4, 4, 5, 6};

auto it = lower_bound(v.begin(), v.end(), 4);
// *it == 4，指向第一个 4

auto it2 = lower_bound(v.begin(), v.end(), 3);
// *it2 == 4，指向第一个 ≥ 3 的元素（即 4）
```

### upper_bound

```cpp
auto it = upper_bound(v.begin(), v.end(), 4);
// *it == 5，指向第一个 > 4 的元素
```

### equal_range

```cpp
auto [lo, hi] = equal_range(v.begin(), v.end(), 4);
// [lo, hi) 包含所有等于 4 的元素
// distance(lo, hi) 就是 4 的个数
```

### binary_search

```cpp
bool found = binary_search(v.begin(), v.end(), 4);
// true

bool found2 = binary_search(v.begin(), v.end(), 3);
// false
```

---

## 什么时候用手动，什么时候用 STL

| 场景 | 推荐 |
|------|------|
| 只需要判断是否存在 | `binary_search` |
| 找插入位置（保持有序） | `lower_bound` |
| 统计某个值出现次数 | `equal_range` |
| 找第一个/最后一个满足条件的 | `lower_bound` / `upper_bound` |
| 面试题要求实现二分 | 手动写 |
| 自定义比较逻辑 | 手动写或传比较器给 STL |

## 自定义比较器

STL 版本都支持自定义比较：

```cpp
// 降序数组上二分
vector<int> v = {6, 5, 4, 3, 2, 1};
auto it = lower_bound(v.begin(), v.end(), 4, greater<int>());

// 结构体按某个字段二分
struct Person { int age; string name; };
vector<Person> people = {{20, "a"}, {25, "b"}, {30, "c"}};

auto it = lower_bound(people.begin(), people.end(), 25,
    [](const Person& p, int val) { return p.age < val; });
```

---

## 踩坑点

- ❌ 在**未排序**的数组上用 `lower_bound` — 结果是未定义的
- ❌ 用 `lower_bound` 找目标值，但数组里有重复元素 — 返回的是第一个 ≥ val 的，不一定是目标值本身
- ❌ 忘记检查返回值是否等于 `end()` — 没找到时返回 end
- ❌ 手动实现时写 `(left + right) / 2` — 大数溢出
- ✅ `lower_bound` 返回的是迭代器，需要解引用或计算距离

## 延伸

- `std::partition_point` — C++11，对按条件划分的范围做二分
- `std::ranges::lower_bound` — C++20，支持 ranges 管道语法
