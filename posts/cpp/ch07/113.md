# `std::print` / `std::println` (C++23)

C++23 将格式化与输出合二为一，`println` 自动追加换行，可直接指定输出目标（`stdout`/`stderr`），告别 `std::cout << std::format(...)` 的冗长。

| 写法 | 标准 | 换行 | 目标指定 | 简洁度 |
|------|------|------|---------|--------|
| `printf(...)` | C | 手动 `\n` | 隐式 stdout | 中 |
| `cout << format(...)` | C++20 | 手动 | 固定 cout | 冗长 |
| `std::println(...)` | C++23 | **自动** | 可指定 | 简洁 |

```cpp
#include <print>  // C++23

int main() {
    std::print("Hello {}\n", "World");           // 输出到 stdout
    std::println("Value: {}", 42);               // 自动换行
    std::println(stderr, "Fatal: {}", "oom");   // 输出到 stderr
    
    // 复用 format 的格式说明
    std::println("{:>10.2f}", 3.14);             // "      3.14"
}
```

**原理解析 + 常见误区**

- ❌ `std::print` 是全新实现与 `format` 无关 — 错误。底层基于 `std::format` + 操作系统原生 API。
- ❌ C++23 已全面普及 — 错误。目前 GCC 13+、Clang 17+、MSVC 2022 部分支持，需注意编译器版本和 `-std=c++23` 标志。

**延伸点：** `std::print` 底层通常基于 `std::format` + `std::FILE*` 或操作系统原生 API，性能优于 `cout`。需注意编译器版本支持情况。
