# Fold Expressions（折叠表达式）

C++17 引入了折叠表达式，让变参模板的展开从递归写法变为**一行语法**。

---

## 一、为什么要折叠

C++11/14 对变参求和需要递归：

```cpp
// C++11 递归展开
auto sum() { return 0; }
template<typename T, typename... Args>
auto sum(T first, Args... rest) {
    return first + sum(rest...);
}
```

C++17 折叠表达式一行搞定：

```cpp
template<typename... Args>
auto sum(Args... args) {
    return (args + ...);    // 一元右折叠
}
```

---

## 二、四种折叠形式

| 形式 | 语法 | 展开示例（对 `args = {1, 2, 3, 4}`） |
|------|------|--------------------------------------|
| 一元右折叠 | `(args op ...)` | `1 op (2 op (3 op 4))` |
| 一元左折叠 | `(... op args)` | `((1 op 2) op 3) op 4` |
| 二元右折叠 | `(args op ... op init)` | `1 op (2 op (3 op (4 op init)))` |
| 二元左折叠 | `(init op ... op args)` | `(((init op 1) op 2) op 3) op 4` |

```cpp
// 一元右折叠（从右向左结合）
template<typename... Args>
auto sum_right(Args... args) {
    return (args + ...);
}

// 一元左折叠（从左向右结合）
template<typename... Args>
auto sum_left(Args... args) {
    return (... + args);
}

// 对于可交换的运算（加法、乘法），左右结果相同
// 对于减法等不可交换的运算，结果不同！
std::cout << sum_right(1, 2, 3, 4);  // 1 + (2 + (3 + 4)) = 10
std::cout << sum_left(1, 2, 3, 4);   // ((1 + 2) + 3) + 4 = 10
```

## 三、常用操作示例

### 打印所有参数

```cpp
template<typename... Args>
void print_all(Args... args) {
    ((std::cout << args << " "), ...);   // 逗号运算符折叠
}
print_all(1, "hello", 3.14);   // 输出: 1 hello 3.14
```

### 检查是否全部满足条件

```cpp
template<typename... Args>
bool all_true(Args... args) {
    return (... && args);    // 从左到右 && 折叠
}
all_true(true, true, false);  // false
```

### 检查是否存在

```cpp
template<typename... Args>
bool any_zero(Args... args) {
    return (... || (args == 0));
}
any_zero(1, 2, 0, 3);  // true
```

### 元组遍历

```cpp
template<typename... Args>
void for_each(std::tuple<Args...>& tup) {
    std::apply([&](auto&... args) {
        ((std::cout << args << " "), ...);
    }, tup);
}
```

## 四、支持的运算符

几乎所有二元运算符都支持折叠：

| 类别 | 运算符 |
|------|--------|
| 算术 | `+` `-` `*` `/` `%` |
| 比较 | `==` `!=` `<` `<=` `>` `>=` |
| 逻辑 | `&&` `\|\|` |
| 位运算 | `&` `\|` `^` `<<` `>>` |
| 赋值 | `=` `+=` 等（通常和 lambda 配合） |
| 逗号 | `,` |

---

## 五、常见误区

- ❌ 折叠表达式只能用于变参模板 — ✅ 对，折叠仅对参数包有效。
- ❌ 一元折叠需要非空包 — ✅ 如果包为空且运算符不支持空包，会编译错误。逻辑 `&&` 空包为 `true`，`||` 空包为 `false`，逗号空包为 `void()`。
- ❌ 左折叠和右折叠在所有运算符下结果相同 — ✅ 对可交换运算相同，对减法/除法/移位等不可交换运算**不同**。
- ❌ 折叠表达式只能出现在函数返回值位置 — ✅ 可以出现在任何需要表达式的上下文：模板实参、`static_assert`、初始化器等。

---

## 六、一句话总结

> 折叠表达式用 `(args op ...)` 或 `(... op args)` 一行代替递归展开，是 C++17 变参模板的语法糖精华。
