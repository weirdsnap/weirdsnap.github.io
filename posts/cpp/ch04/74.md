# 可变参数模板

C++11 通过 `typename... Ts` 声明模板参数包、通过 `args...` 执行包展开；C++17 折叠表达式消除了递归展开的需求。

**场景示例：**
```cpp
// C++17 折叠表达式
template<typename... Ts>
auto sum(Ts... args) {
    return (0 + ... + args);  // 一元右折叠
}
```

**原理解析 + 常见误区**

- ❌ 参数包可以像数组一样索引访问 — 错误。C++ 不支持 `args[0]` 或 `Ts[1]`。
- ❌ 可变参数函数模板不需要终止条件 — 错误。C++11 递归展开必须提供零参数的重载作为终止条件。

**延伸点**：C++17 `if constexpr` 替代递归终止：通过编译期分支在单函数内处理空包与非空包，无需编写独立的终止重载。
