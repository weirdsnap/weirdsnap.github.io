# std::atomic

`std::atomic` 提供对单个数据类型的原子读写操作，保证多线程并发访问时的数据竞争自由。

| 操作 | 说明 |
|------|------|
| `load()` | 原子读 |
| `store()` | 原子写 |
| `exchange()` | 读写交换旧值 |
| `compare_exchange_weak` | 可能伪失败，适合循环 |
| `compare_exchange_strong` | 保证真失败，适合单次检查 |

**原理解析 + 常见误区**

- ❌ `std::atomic` 对象可以拷贝 — 错误。`atomic` 的拷贝构造和拷贝赋值被删除。
- ❌ `counter = counter + 1` 是原子的 — 错误。这是先 `load`、再计算、再 `store` 的三步操作。

**延伸点**：C++20 `std::atomic_ref` 允许对非 `atomic` 对象（如共享内存中的裸变量）执行原子操作。
